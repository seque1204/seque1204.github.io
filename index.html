Projects

Social Security Analysis Project: "Evaluating Safety and Security in Costa Rica Using Synthetic Controls"

I developed this project to better understand the impact of socio-political factors on safety and security in Costa Rica. Motivated by my interest in data-driven policy evaluation, I explored how changes in governance and other variables influence societal safety. This analysis uses regression discontinuity methods and synthetic control modeling to measure the effects of thresholds, such as policy shifts, on safety indicators. In Python, I prepared and visualized data using Weighted Least Squares (WLS) regressions. To complement this, I employed R's Synth package to construct a counterfactual "Synthetic Costa Rica," comparing it to the real data to evaluate the effect of specific interventions over time. The project highlights the power of combining advanced statistical techniques with visualization tools to uncover insights from real-world data. By merging Python and R, I was able to conduct a robust analysis that offers actionable insights for improving public safety policies. This experience strengthened my skills in multi-tool data analysis, regression methods, and effective storytelling with data.

UFC Predictions Project: "Predicting UFC Fight Outcomes with Machine Learning"

I created this project out of a deep interest in mixed martial arts and my desire to combine that passion with my growing skills in data science and machine learning. As a fan of the UFC, I’ve always been fascinated by the dynamics of fight outcomes. But predicting winners isn’t just about intuition—it's about uncovering patterns in the data. The goal of this project was to build a machine learning model capable of predicting the outcomes of UFC fights based on historical fight data and fighter statistics. I started by gathering and preprocessing a dataset, performing feature engineering to highlight meaningful factors, and training a neural network model to analyze the data. Along the way, I explored visualization techniques to better understand the learning process and refined the model to achieve over 70% prediction accuracy. This project wasn’t just about creating a functional prediction model—it was about developing a deeper understanding of machine learning concepts and seeing how these technologies can be applied to real-world problems. Through this experience, I’ve gained insights into the importance of data preprocessing, model evaluation, and iterative improvement. Ultimately, this project allowed me to merge my interests in sports analytics and programming, demonstrating how data-driven approaches can provide a new perspective on something as unpredictable as a UFC fight.
